🚀 ISP Billing & Management System

A full-featured ISP Billing and Customer Management System designed for small to medium-scale internet service providers. This system integrates with MikroTik PPPoE, supports automated billing, reminders, and future-ready payment integrations.

📌 Features
👥 Customer Management
Add / Edit / Delete clients
PPPoE credentials handling
Installation date & due date tracking
Account status (Active, Disabled, Pending)
💰 Billing System
Monthly billing cycles
Due date monitoring
Automated reminders (planned via API)
Payment tracking
📡 MikroTik Integration (Planned / Partial)
PPPoE user management
Dynamic IP handling
Session monitoring
Redirection for unpaid users
📊 Admin Dashboard
Total customers overview
Verified accounts tracking
Pending verification alerts
Recent customers list
🌐 Web-Based System
Built with Laravel (backend)
Responsive UI (mobile-friendly)
Admin panel + client-side expansion ready
